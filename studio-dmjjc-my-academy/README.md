# Del Mar Jiu-Jitsu Club Sanity Studio

Sanity Studio for editing Community Hub content.

Project ID: `z48r70x2`
Dataset: `production`

## Run

```bash
npm install
npm run dev
```

Open the local Studio URL printed in the terminal, usually `http://localhost:3333`.

## Content Types

- Announcements
- Events
- Videos
- Site Status
